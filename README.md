A world-class mobile app for the dedicated Murph athlete.

## Features

- **Murph Wizard**: Auto-scale your Murph based on percentage (0-100%) and Vest weight.
- **Smart Recommendations**: Weekly analysis of your logs to suggest safe progressive overload (+10% rule).
- **Neon Dark Mode**: Beautiful, high-contrast aesthetic.
- **Offline First**: Built with SQLite.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npx expo start
   ```

## Tech Stack

- React Native (Expo SDK 50)
- Expo Router
- NativeWind (TailwindCSS)
- SQLite
- React Native Reanimated

## Workouts

The app follows the mandated weekly schedule:
- **Mon**: Scaled Murph
- **Tue**: Heavy Day
- **Wed**: Recovery
- **Thu**: Zone 2 Run
- **Fri**: Murph Time Trial
- **Sat**: Rucking
- **Sun**: Rest
